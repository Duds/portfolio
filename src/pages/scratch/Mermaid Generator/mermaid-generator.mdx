# Mermaid Generator

An interactive tool for creating and visualizing Mermaid diagrams in real-time. This project explores the integration of Monaco Editor with Mermaid.js to create a powerful diagramming tool.

## Project Goals

- Create a standalone Mermaid diagram editor
- Integrate Monaco Editor for a better editing experience
- Support real-time preview of diagrams
- Enable export of diagrams in various formats
- Provide a clean, intuitive user interface

## Technical Exploration

- Monaco Editor integration for syntax highlighting and editing
- Mermaid.js for diagram rendering
- TypeScript for type safety
- Astro for static site generation
- TailwindCSS for styling

## Current Status

The project is currently in active development. The basic editor and preview functionality are working, with ongoing improvements to the user interface and additional features.
